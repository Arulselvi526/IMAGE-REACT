# 📸 Image Gallery React App

A responsive React-based Image Gallery that showcases beautiful images using reusable components, dynamic rendering with map(), and a clean, user-friendly interface. Designed with a responsive layout to provide a seamless experience across all devices.

🔗 Live Demo: https://image-react-tau.vercel.app/

🔗 GitHub Repository: https://github.com/Arulselvi526/IMAGE-REACT

## 🚀 Features

- 📸 Modern photo gallery with responsive design
- ⚛️ React component-based development
- 🔄 Dynamic content rendering from a data array
- 🧩 Reusable Image Card components
- 📤 Efficient data sharing using Props
- 📱 Optimized for mobile, tablet, and desktop
- 🎨 Clean, intuitive, and visually appealing interface
- ⚡ Lightweight and high-performance application
- 📂 Well-structured and maintainable codebase
- 🌟 Easy to extend with additional images and features

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
